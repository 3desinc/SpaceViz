Configuring git on windows:
   1. download git Bash from gitforwindows.org
   2. navigate to folder where you want to clone the repository
   3. git clone https://3desinc:password@github.com/3desinc/SpaceViz.git


Adding a file to a repository [from git bash]:
   1. Enter command -> git add .
   2. Enter command -> git commit -m "file_name"
   3. Enter command -> git push origin master
